# Product FAQ - TechFlow Platform

## General Questions

### What is TechFlow?
TechFlow is an all-in-one project management and collaboration platform designed for modern software teams. It combines task management, code repository integration, CI/CD pipelines, and team communication into a single unified experience.

### What platforms is TechFlow available on?
TechFlow is available as:
- Web application (Chrome, Firefox, Safari, Edge)
- Desktop apps for Windows and macOS
- Mobile apps for iOS and Android
- API for custom integrations

### What is the pricing structure?
- **Free Tier**: Up to 5 users, 3 projects, basic features
- **Team Plan**: $12/user/month, unlimited projects, advanced analytics
- **Enterprise Plan**: Custom pricing, SSO, dedicated support, SLA guarantees

## Features & Functionality

### How do I create a new project?
1. Click the "+" button in the sidebar
2. Select "New Project"
3. Choose a template (Kanban, Scrum, or Custom)
4. Add team members and set permissions
5. Configure integrations (GitHub, Jira, Slack)

### Can I integrate TechFlow with GitHub?
Yes! TechFlow offers native GitHub integration. To set it up:
1. Go to Project Settings → Integrations
2. Click "Connect GitHub"
3. Authorize the TechFlow GitHub App
4. Select repositories to sync
5. Enable automatic issue sync and PR tracking

### How does the AI assistant work?
Our AI assistant, TechFlow AI, can:
- Summarize meeting notes automatically
- Generate task descriptions from requirements
- Suggest task assignments based on team capacity
- Predict project completion dates
- Answer questions about your project data

### What are Workflows?
Workflows are customizable automation rules. Examples:
- Automatically assign QA review when a task moves to "Testing"
- Send Slack notifications when blockers are identified
- Create follow-up tasks when bugs are marked as "Resolved"

## Troubleshooting

### Why are my notifications not working?
Check these common issues:
1. Browser notification permissions are enabled
2. Email notifications are enabled in Settings → Notifications
3. Your email is not in spam folder
4. Push notifications are enabled on mobile

### The app is running slowly. What can I do?
Try these solutions:
1. Clear browser cache and cookies
2. Disable browser extensions temporarily
3. Check your internet connection speed
4. Reduce the number of open projects/tabs
5. Contact support if issues persist

### I can't access a project. What should I do?
This usually means:
- You haven't been added to the project team
- Your account permissions have changed
- The project has been archived
Contact your project admin or workspace owner to resolve access issues.

## Security & Privacy

### Is my data encrypted?
Yes, all data is encrypted:
- At rest: AES-256 encryption
- In transit: TLS 1.3
- Backups: Encrypted and stored in geographically distributed locations

### Where is my data stored?
Data is stored in AWS data centers. Enterprise customers can choose their data region:
- US (Virginia)
- EU (Frankfurt)
- APAC (Singapore)

### Does TechFlow comply with GDPR?
Yes, TechFlow is fully GDPR compliant. We provide:
- Data export functionality
- Right to deletion (account removal)
- Transparent privacy policy
- Data Processing Agreement available on request
